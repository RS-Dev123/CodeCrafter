# CodeCrafter
Axiora-Fake News Detection System @Rupam Sur @Makinur Rahman


AI-Powered Fake News Detection System

🧠 Detect misinformation instantly using AI + NLP
🌐 Built with Python, Flask, and a modern interactive frontend

🌟 Overview

In the age of digital media, fake news spreads faster than ever — influencing opinions, creating panic, and distorting reality.

This project introduces an AI-driven Fake News Detection System that analyzes news content and determines whether it is Real or Fake, along with a confidence score.

✨ Designed for speed, simplicity, and scalability, this system demonstrates how AI can be used to fight misinformation effectively.

🎯 Problem Statement

🚨 Rapid spread of misinformation on social media

🧾 Manual fact-checking is slow and inefficient

🌍 Fake news impacts public trust and decision-making

👉 Goal: Build an automated system to detect fake news using AI techniques.

💡 Solution

We developed a web-based AI system that:

Accepts news text as input

Processes it using NLP-based logic

Classifies it as Fake or Real

Displays a confidence score instantly

⚡ Key Features

✨ AI-powered text analysis
📊 Confidence score output
🌐 Clean and responsive UI
⚡ Fast backend with Flask
🔌 Easily extendable with ML models & APIs

🔹 6. Run Frontend

Open frontend/index.html in browser

Use Live Server in VS Code
